# Правила для режима AI Debugger

Эта директория содержит модули промпта для режима `ai-debugger`.

## Структура

Модули загружаются автоматически при активации режима AI Debugger.

## Использование

Правила из этой директории загружаются автоматически и добавляются к основному промпту.

Для lazy loading используйте MCP tool `mcp--roo-trace--load_rule` для загрузки конкретного модуля по требованию.

## Текущие модули

- `00-formats-validator.md` - Валидация форматов summary от Архитектора и Кодера

## Планируемые модули

Согласно плану в `features/PROMPT_PHASES.md`, будут созданы модули:
- Базовые модули (00-base-*.md)
- RooTrace модули (roo-*.md)
- Архитектор модули (arch-*.md)
- Кодер модули (code-*.md)
